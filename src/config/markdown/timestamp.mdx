Requests to

```bash
https://stampthetime.herokuapp.com/api/:time
```

where `time` is a valid date string, will return an object of the form

```json
{
  "unix": "unix-time-stamp",
  "utc": "utc-date-string"
}
```

```javascript
let foo = bar;

function blah() {
  return 'blah';
}
```

For a current timestamp, simply omit `time` from the URL.
